듣기 오디오 생성기

Генерирует аудио для всех диалогов 듣기 через ElevenLabs API прямо в этом браузере. Ключ никуда не отправляется, кроме официального API ElevenLabs. После генерации получишь ZIP-архив с готовыми MP3 файлами.

Бесплатный лимит ElevenLabs — 10 000 символов в месяц. Этот набор диалогов занимает примерно 1 730 символов, так что укладывается с большим запасом даже с учётом повторных попыток.
